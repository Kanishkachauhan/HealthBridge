# 🏥 HealthBridge - Doctor Appointment System

## 📌 Project Description
HealthBridge is a web-based project that allows users to serach doctors, book appointments with doctors easily. It helps patients and doctors manage schedules efficiently.

---

## 🚀 Features
- 🔐 User Authentication (Login / Register)
- 👨‍⚕️ Doctor Listing
- 📅 Appointment Booking
- 🧾 User Dashboard
- ⚙️ Admin Panel (optional)

---

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Other: CSS, JavaScript
